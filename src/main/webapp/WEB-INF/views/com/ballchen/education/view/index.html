学习计划

前言

什么是基本功

想用钢琴弹出优美的乐曲，基本功是必须要学习的。所谓基本功就是正确的弹琴方法。

学习周期

关于基本功学习周期

基本功的学习周期按“月”划分。


大部分进度正常的同学通过4个月的学习可以完成第一个阶段的课程。


如果中间出差不能连续学习也可以暂停。


如何学习

通过网络如何学习钢琴

通过网络学习，有效的降低了我们的学习成本，我们的学习理念是每天学习一点点。


基本功的学习是需要积累的，每天学习1个知识点，每天练习一首简单的练习曲。


每天发作业给老师，每天去解决一个问题。


学习效率会比传统面授课程高的多。

在哪学习

关于我们的教学安排

每个付费学员都会拥有一个班级，每个班级不超过40人。


老师通过QQ群视频授课，集体授课，1对1解答同学的问题，回复同学的作业。


如果落下了直播课也有录播课程可以观看。只要有时间就能随时随地学钢琴。

关于学费

我们的收费制度

学费每个月100元-200元不等。


1个月的高品质学习付出的费用只等于线下钢琴老师1节课的课时费。

定期活动

我们平时的集体活动

每周末除了正常的课程和作业回复外，周末会进行周学习总结。


把一周学习的重点内容更细致的剖析复习。


月末进行汇报演出，让同学试胆，避免弹琴见光死的悲剧。

学习效果

通过学习，达到的效果

经过4个月的课程，大部分同学都能够在识谱、手型等方面获得较系统的学习。


具备自学大部分非炫技的流行歌曲、流行钢琴曲并用正确的方式演奏出来的能力。


家里的钢琴终于不是摆设了，也圆了自己从小的钢琴梦。
